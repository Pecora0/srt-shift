# srt-shift

Tool for editing the time in SubRip text files (.srt)

## Quick start

Download the `srt-shift` and run it using lua (you might need to adjust the first line).
If you have a file `input.srt` and want every subtitle block appear and disappear two seconds **later** and write the result to `output.srt`
use the following command

```console
$ srt-shift -d 2000 input.srt -o output.srt
```
